Preguntas y respuestas de laboratorio

Preguntas lab 17

  • ¿Qué ventajas tiene escribir el código SQL únicamente en la capa del modelo?

    • Porque estructuramos mejor el proyecto, podemos tener mayor claridad de en donde manejar los datos y si hay un error en cuanto a esta interacción nos brinda una mayor facilidad para detectarlo

  • ¿Qué es SQL injection y cómo se puede prevenir?

    • Es un tipo de atace a sistemas de información en que se busca introducir código SQL en campos de entrada y que sea ejecutado, existen diferentes formas de prevenirlo, en las que se especifican parametros y validaciones al obtener y almacenar información o métodos específicos para interactuar con bases de datos
