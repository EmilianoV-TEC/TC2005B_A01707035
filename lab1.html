Preguntas

  • ¿Cuál es la diferencia entre Internet y la World Wide Web?

    • El internet se refiere a una red de computadoras que existe alrededor del mundo, mientras que la World Wide Web indica la red de páginas que se encuentran en el internet

  • ¿Cuáles son las partes de un URL?

    • El protocolo, el dominio, la ruta y los parámetros

  • ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

    • GET: Solicitar datos de un servidor
    HEAD: Permite revisar si cierta información está disponible en el servidor
    POST: Enviar datos a un servidor
    PUT: Actualiza cierta información o recurso publicado en el servidor
    PATCH: Permite modificar un recurso
    DELETE: Para borrar un recurso en el servidor

  • ¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

    • El método POST ya que este se encarga de enviar datos al servidor y no modifica el URL con información importante

  • ¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?

    • El método GET ya que se está solicitando la información del sitio que se va a desplegar
