Preguntas lab 4

  • ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

    • Javascript no tiene tipado y esté enfocado en el desarrollo de aplicaciones o páginas web, Java es un lenguaje tipado, compilando y se puede usar en varias herramientas. Ambos surgen del lenguaje C

  • ¿Qué métodos tiene el objeto Date?

    • getFullYear, getMilliSeconds, getHours, setYear, toString

  • ¿Qué métodos tienen los arreglos?

    • push, join, toString, pop, splice

  • ¿Cómo se declara una variable con alcance local dentro de una función?

    • Con la palabra reservada let, seguida por el nombre de la variable

  • ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    • Que esta variable puede ser utilizada y editado por otras partes del sistema o programa. Este comportamiento puede que no sea deseado y llega a ser inseguro
